# EduTask Manager

## Project Description
Student Task Manager is a modern web application designed to help students organize, track, and manage their academic tasks and subjects efficiently. The dashboard provides real-time statistics, achievements, and visual analytics to boost productivity and keep students on track throughout the semester.

## Features
- Dashboard with live statistics and achievement tracking
- Task CRUD (Create, Read, Update, Delete) with filtering and sorting
- Subject management with dynamic stats and card grid
- Responsive design for desktop, tablet, and mobile
- Notification bar and profile dropdown navigation
- Sidebar navigation with overlay for mobile/tablet
- LocalStorage persistence for tasks and subjects
- Chart.js analytics for task completion and study hours
